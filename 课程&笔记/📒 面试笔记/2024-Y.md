## Aliases

```dataview
TABLE aliases
FROM "BEFOREJournal"
WHERE aliases != null
WHERE length(aliases) > 1
WHERE file.day.year = 2024
```

## Highlights

```dataview
TABLE WITHOUT ID dateformat(file.ctime, "yyyy-MM") AS Month, file.day.weekyear AS Week, highlights
FROM "Journal"
WHERE highlights != null
WHERE file.day.year = 2024
SORT file.day.weekyear
```
